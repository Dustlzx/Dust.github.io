luckyexcel 引入教程 ​

luckyexcel 文档地址：快速上手 | Luckysheet 文档 (gitee.io)

简介

luckyexcel 使得 vue 实现在线的编辑表格功能

luckyexcel 引入教程 ​

  1. 从 git 上拉取代码

    • 源码地址：

      javascript
      https://gitee.com/mengshukeji/Luckysheet#https://mengshukeji.github.io/LuckysheetDemo

    • 拉取代码到本地

      javascript
      git clone https://gitee.com/mengshukeji/Luckysheet.git

  2. 进入项目中使用 yarn 安装依赖

  3. 运行 yarn build 打包项目

  4. 把打包好的 dist 文件夹下的文件移动到我们的 public 文件下（也就是跟 index.html 文件同目录下）

  5. 在 index.html 文件中引入 js

    javascript
    <link rel='stylesheet' href='./plugins/css/pluginsCss.css' />
<link rel='stylesheet' href='./plugins/plugins.css' />
<link rel='stylesheet' href='./css/luckysheet.css' />
<link rel='stylesheet' href='./assets/iconfont/iconfont.css' />
<script src="./plugins/js/plugin.js"></script>
<script src="./luckysheet.umd.js"></script>

  6. 在页面中使用

    javascript
    <div
  id='luckysheet'
  style='margin:0px;padding:0px;position:absolute;width:100%;height:90%;left: 0px;top: 50px;'
></div>
    javascript
    onMounted(() => {
  //配置项
  const luckysheet = window.luckysheet;

  var options = {
    container: 'luckysheet', //luckysheet为容器id
    title: '任务清单', // 设定表格名称
    lang: 'zh' // 设定表格语言
  };
  luckysheet.create(options);
});

问题： ​

1. npm 包 没有声明文件 类型“Window & typeof globalThis”上不存在属性 ​

​ 方法 1：

javascript
// 在 node_modules\@types 下 添加npm为名称的 .d.ts文件
declare module 'luckyexcel'

declare interface Window {
    luckysheet: any
}

​ 方法 2：

在 env.d.ts 下添加代码：

javascript
declare interface Window {
    luckysheet: any
}
 declare module 'luckyexcel'
